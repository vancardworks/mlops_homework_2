API Documentation

Overview

This API allows you to analyze the sentiment of text by sending a POST request to the endpoint.

API Endpoint

The API is accessible at: http://localhost:3000/api/v1/analysis/

Request

To analyze a bit of text, send a POST request with a JSON object containing a single field "text" pointing to the image to be analyzed. 

            
{
    "text": "some_text"
}
